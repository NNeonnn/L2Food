Проект сделан командой "Не хухры-мухры", состоящей из Артимовича Кирилла, Беззубцева Ивана, Кузнецова Дмитрия, Пушкарёва Евгения и Ушакова Дениса

Это веб-приложение для управления столовой, на данный момент в процессе разработки. Данное веб-приложение
должно автоматизировать процесс оформления и оплаты заказов учениками и их родителями.

Стек: python, flask, docker, mongoDB (для управления аккаунтами), sqlite (в процессе, происходит переход на sqlite с json)

Инструкции к установке и запуску (и дебагу) для разработчиков (на данный момент запускать можно только через VSCode, чтобы дебаг работал):
- Установите Docker Engine и docker-compose (В github codespace должны быть по умолчанию)
- Запустите контейнер через docker compose up --build
- Если нужен режим дебага в вскоде: Нажмите F5 в VSCode, чтобы подсоединиться к дебаггеру (когда в консоли появятся первые логи, начинающиеся на "web-1  | ")
- Откройте [http:](http://127.0.0.1:5237/) в браузере.
- Чтобы выключить напишите docker-compose down или дважды прожмите ctrl + C
